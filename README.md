## 🚀 How to Use

1. **Choose your dashboard** based on your needs:
   - Start with **Teams Dashboard** for quick team-based overview
   - Use **Complete Dashboard** for detailed analysis and dependency mapping

2. **Use interactive features:**
   - Filter by team (Teams Dashboard)
   - Search by project key or name
   - Filter by status, priority, or effort
   - View epic status to track child initiative progress

3. **Share the URLs:**
   - Send specific dashboard links to relevant teams
   - Use the filtered view for focused discussions

4. **Export data (Complete Dashboard):**
   - Export to JSON for data analysis
   - Export to CSV for spreadsheet work
